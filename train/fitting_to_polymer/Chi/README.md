This directory contains the data and script to compare experimental Chi values with Chi values computed from COSMO calculations.

#### 1. `./data/reference_chi_canon.csv`
Contains extracted experimental Chi values.

#### 2. `./chi_results/chi_prediction_0_025_solute_toluene_cosmo_solvent_cosmo_cosmo.csv`
Contains Chi values computed using COSMO-SAC calculations.

#### 3. `./SI_figure.py`
Generates a plot showing the correlation between experimental Chi values and COSMO-SAC-predicted Chi values.
